# AgendaEvaluacion
Esta aplicacion permite realizar varias funciones como añadir la informacion (nombre y teléfono) de varios contactos, mostrarlos en una lista y borrarlos.

Permite guardar los datos de los contactos creados en un fichero de texto cuando se detiene la aplicación y cargarlos cuando se inicia.

Contiene una lista de comandos de ayuda para escribir:

* A(add):Añade un nuevo contacto.
* L(list):Muestra la lista de contactos.
* D(delete):Borra de la lista un contacto.
* Q(quit):Sale de la aplicación.
